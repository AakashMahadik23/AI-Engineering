Features :- 
Smart link extraction & filtering
Clean website content scraping
AI-powered relevance detection (LLM)
Automatic brochure generation in Markdown
Runs fully locally using Ollama
Lightweight & extensible pipeline

Installation & Setup :- 
Clone Repository
git clone https://github.com/your-username/ai-brochure-generator.git
cd ai-brochure-generator

Install Dependencies :- 
pip install -r requirements.txt

Or manually:
pip install requests beautifulsoup4 python-dotenv openai

Install Ollama :- 
Download from https://ollama.com/

Pull the model:
ollama pull llama3.2

Run Ollama:
ollama serve
